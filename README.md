**Command System Emulator PDP-11** — это кроссплатформенный эмулятор классической архитектуры PDP-11 с графическим интерфейсом. Проект создан в образовательных целях для изучения архитектуры ЭВМ, ассемблера и принципов работы процессора.

Эмулятор написан на **C++17** с использованием фреймворка **Qt6**, поддерживает "на лету" дизассемблирование машинного кода, пошаговую отладку и работу с виртуальными внешними устройствами (дисплей, клавиатура, принтер).

---

## ✨ Ключевые возможности

*   **Интерактивная память и регистры:** Просмотр и редактирование содержимого оперативной памяти и регистров процессора (R0-R7, PSW) в реальном времени.
*   **Встроенный дизассемблер:** Автоматический перевод введенных восьмеричных машинных кодов в мнемоники ассемблера PDP-11 прямо в таблице памяти.
*   **Режимы выполнения:** 
    *   `Step Mode` — пошаговое выполнение для детальной отладки.
    *   `Program Mode` — непрерывное выполнение.
*   **Виртуальные внешние устройства (Memory-Mapped I/O):**
    *   Клавиатура (TKS `177560`, TKB `177562`)
    *   Терминал / Дисплей (TPS `177564`, TPB `177566`)
    *   Принтер (`177514`, `177516`)
    *   Аппаратный таймер с генерацией прерываний по вектору `100(8)`.
*   **Динамическая тема оформления:** Автоматическая поддержка светлой и темной темы в зависимости от настроек вашей операционной системы (Windows / Linux).
*   **Справочная система:** Встроенная контекстная справка по командам и прилагаемый подробный справочник в формате PDF.
*   **Сохранение и загрузка:** Поддержка импорта и экспорта программ в формате `.pdp`.

## 🚀 Примеры программ

В папке `Programs/` находятся готовые программы для тестирования эмулятора (подробности в файле [Program Descriptions.txt](Programs/Program%20Descriptions.txt)):

1.  **Hello World** — классический вывод строки на экран терминала с использованием цикла и проверки флага готовности устройства.
2.  **Keyboard** — эхо-ввод: считывание символов с клавиатуры и их вывод на экран с выходом по нажатию клавиши 'q'.
3.  **Print** — вывод текста на виртуальное устройство печати.
4.  **Single-digit Adder** — программа, принимающая с клавиатуры выражение вида `A+B`, вычисляющая сумму через ASCII-коды и выводящая результат (например, `3+4=7` или `6+5=11`).
5.  **16-bit Integer Calculator** — полноценный калькулятор с поддержкой сложения и вычитания многозначных чисел (использует подпрограммы деления и конвертации ASCII).

## 🛠️ Сборка и установка

### 🐧 Сборка под Linux (Arch Linux / Manjaro)

1. Установите необходимые зависимости (компилятор, CMake, Qt6):
   ```bash
   sudo pacman -S base-devel cmake qt6-base mesa
   ```
2. Склонируйте репозиторий и запустите скрипт сборки:
   ```bash
   git clone https://github.com/Arta48/PDP11.git
   cd PDP11
   sh compile.sh
   ```
3. Запуск эмулятора:
   ```bash
   ./build/PDP11
   ```

### 🪟 Сборка под Windows

Проект включает скрипт для полностью автоматизированной сборки независимого `.exe` файла, который не требует предварительной настройки окружения.

1. Запустите файл `compile.bat` (двойным кликом или через консоль).
2. Скрипт всё сделает за вас:
   * Скачает и установит среду **MSYS2** (в `C:\msys64`).
   * Скачает компилятор GCC, CMake и статическую версию Qt6 со всеми зависимостями.
   * Сконфигурирует и скомпилирует проект.
3. Готовый к работе `PDP11.exe` появится в директории `build-windows/`.

*💡 Примечание: Для полной очистки системы от среды сборки после окончания компиляции достаточно просто удалить папку `C:\msys64`.*

### 🐧 Кросс-компиляция для Windows (из-под Linux)

Проект поддерживает сборку статического `.exe` файла для Windows напрямую из Linux через кросс-компиляцию MinGW.

1. Установите MinGW и добавьте репозитории `ownstuff`:
   ```bash
   sudo pacman -S mingw-w64-gcc
   
   echo -e "\n[ownstuff]\nSigLevel = Optional TrustAll\nServer = https://ftp.f3l.de/~martchus/\$repo/os/\$arch\nServer = https://martchus.dyn.f3l.de/repo/arch/\$repo/os/\$arch" | sudo tee -a /etc/pacman.conf
   
   sudo pacman-key --keyserver keyserver.ubuntu.com --recv-keys B9E36A7275FC61B464B67907E06FE8F53CDC6A4C
   sudo pacman-key --finger B9E36A7275FC61B464B67907E06FE8F53CDC6A4C
   sudo pacman-key --lsign-key B9E36A7275FC61B464B67907E06FE8F53CDC6A4C
   
   sudo pacman -Syy
   sudo pacman -S mingw-w64-cmake mingw-w64-qt6-base-static
   ```
2. Запустите скрипт сборки для Windows:
   ```bash
   sh compile_windows.sh
   ```
3. Готовый файл `PDP11.exe` будет лежать в директории `build-windows/`.

## 📁 Структура проекта

*   `src/` — исходный код на C++ (`Pdp11.cpp` - ядро эмулятора, `MainWindow.cpp` - графический интерфейс).
*   `assets/` — ресурсы графического интерфейса (иконки, шрифты для UI).
*   `Programs/` — коллекция готовых программ (дампы `.pdp`) и их описание.
*   `CMakeLists.txt` — конфигурация сборочной системы.
*   `PDP11.pdf` — справочное руководство по архитектуре и системе команд (копируется в папку сборки).
